Que contient BOOTSTRAP ?

C'est un ensemble de :

  • Composants HTML : card, forms, modal..
  • Classes CSS : display, flex, text..
  • Plugin Javascript : events, transitions..
  • D'outils : préprocesseurs CSS (LESS, SASS), npm..
